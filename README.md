# Event Management System

## Project Overview
The Event Management System is a web-based application designed to simplify the process of organizing and managing events. It allows users to view events, book tickets, and access event-related information easily. Administrators can manage events, attendees, and system data efficiently.

---

## Purpose
The main purpose of this project is to:
- Reduce manual effort in event management  
- Provide an easy platform for booking events  
- Maintain organized and secure data  
- Improve efficiency in handling events  

---

## Features
- View available events  
- Check event details  
- Book tickets  
- Contact organizers  
- View gallery  

---

## User Roles
- **User:** View events, book tickets, contact organizers  
- **Admin:** Manage events, attendees, and reports  

---

## Technologies Used
- Frontend: HTML, CSS  
- Backend: PHP  
- Database: MySQL  

---

## Database Structure
The system uses a MySQL database with the following main tables:

- **Events Table:** Stores event details (name, date, location, description)  
- **Users Table:** Stores user information  
- **Bookings Table:** Stores ticket booking details  
- **Admin Table:** Stores admin login and control data  

---


## How to Run
1. Install XAMPP  
2. Place project folder in `htdocs`  
3. Import `database.sql` into phpMyAdmin  
4. Start Apache and MySQL  
5. Open browser and go to:
   http://localhost/event-management-system
